# argo-dns
The Argo DNS Server
# features
- Authoritative DNS
- Recursive DNS
- DNS Forwarding
- DNS over UDP/TCP
- DNS over TLS
- DNS over HTTPS
- REST API for dynamic configuration
- Database back-end
- Reads BIND Zone files
# supported DBs
- SQLite 
- Mysql
- Postgres
- MongoDB
# supported record types
- A
- AAAA
- AFSDB
- CAA
- CERT
- CNAME
- CSYNC
- DHCID
- DNAME
- EUI48
- EUI64
- HINFO
- SVCB
- HTTPS

- NS
